# 微信小程序跳一跳游戏

这是一个基于微信小程序开发的"跳一跳"小游戏，具有以下特色功能：

## 游戏特色

- 🎮 **简单易玩**：点击屏幕控制小人跳跃
- 🎨 **精美画面**：渐变背景、云朵动画、粒子效果
- 🏆 **计分系统**：实时计分和最高分记录
- ✨ **动画效果**：跳跃粒子、着陆特效、角色动画
- 📱 **响应式设计**：适配不同屏幕尺寸

## 游戏玩法

1. 点击屏幕让红色小人跳跃
2. 小人需要跳到下一个平台上
3. 每成功跳跃一次得1分
4. 掉出屏幕游戏结束
5. 挑战你的最高分记录！

## 技术实现

- **Canvas绘图**：使用Canvas API绘制游戏画面
- **物理引擎**：实现重力和碰撞检测
- **动画系统**：60FPS流畅动画
- **粒子效果**：跳跃和着陆特效
- **数据存储**：本地存储最高分

## 开发说明

1. 使用微信开发者工具打开项目
2. 在`project.config.json`中配置你的AppID
3. 点击编译运行即可体验游戏

## 游戏截图

游戏包含以下视觉元素：
- 天空渐变背景
- 移动的云朵
- 木质平台（带阴影和纹理）
- 红色小人角色（带表情动画）
- 跳跃和着陆粒子效果
- 实时分数显示
- 游戏结束界面

## 自定义配置

你可以在`game.js`中调整以下参数：
- `gravity`: 重力大小
- `jumpPower`: 跳跃力度
- `gameSpeed`: 游戏速度
- 平台生成参数
- 粒子效果参数

## 注意事项

- 游戏需要微信小程序环境运行
- 建议在真机上测试以获得最佳体验
- 可以添加音效和更多动画效果来增强体验
